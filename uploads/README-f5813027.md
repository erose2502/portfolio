# Rhema — Biblical Character AI Platform

Rhema is a mobile-first spiritual companion that pairs users with biblical characters, devotionals, and community features to support daily discipleship. The platform blends AI-guided conversations with Scripture study, prayer journaling, games, and real-time community interactions in a glassmorphic, app-like experience.

## Core Purpose

- Provide faith-centered guidance through conversational AI modeled on biblical characters.
- Create daily spiritual rhythms with Scripture reading, devotionals, and prayer tracking.
- Foster community and accountability with social interactions, multiplayer games, and live calls.

## Architecture Flow (High Level)

1. **User Interface (React)** renders screens, navigation, and PWA experiences.
2. **Feature Components** (chat, prayer journal, games, reading plans, music) manage local UI state and view composition.
3. **Service Layer** handles side effects (auth, calls, storage, realtime events) using Supabase clients.
4. **Supabase Backend** provides Auth, Postgres, Realtime, and Storage.
5. **Edge Functions (Hono)** issue scoped tokens (LiveKit), integrate external APIs (YouTube search), and support AI/voice workflows.
6. **External Providers** (OpenAI, LiveKit, Deepgram) supply AI responses, video calls, and speech features.

## Stack Layers

**Presentation Layer**
- React 18 + TypeScript
- Vite build tooling
- Tailwind CSS v4 + Radix UI primitives + shadcn/ui components
- Lucide icons

**Experience Layer**
- Glassmorphic UI system
- PWA assets and service worker
- Mobile-first navigation patterns

**Domain/Feature Layer**
- Character chat, prayer journal, Bible reader, devotionals, reading plans
- Multiplayer Scripture games and leaderboards
- Community hub, direct messaging, and invitations
- Audio/video calling UI with LiveKit

**Data & Realtime Layer**
- Supabase Auth, Postgres, Storage
- Supabase Realtime channels for presence, calls, and live updates
- Edge Functions (Hono) for tokens and external API access

## Core Features

- **Biblical Character Chats** with multilingual AI responses and persistent history
- **Prayer Journal** with reflections and guided prompts
- **Bible Reader + Reading Plans** for structured study
- **Daily Devotionals** for consistent spiritual practice
- **Scripture Games** with single and multiplayer modes
- **Community Hub** with direct messages and invitations
- **Live Audio/Video Calls** using realtime signaling + LiveKit
- **Music + Media** experiences with curated playback
- **Onboarding & Profiles** with personalization and language settings

## Local Development

1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`

## Configuration Notes

- Supabase credentials are required for auth, realtime, and storage.
- Edge Functions expect secrets like `OPENAI_API_KEY` and LiveKit keys (see `src/DEPLOYMENT_GUIDE.md`).
